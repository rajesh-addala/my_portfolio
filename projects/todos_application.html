Built a Dynamic Todos Application using JavaScript, focusing on DOM manipulation, event handling, and client-side storage. The app allows users to create, manage, and remove todos both statically and dynamically.


Key topics covered:

  • DOM Manipulations: createElement(), appendChild(), classList.add(), classList.remove(), classList.toggle().

  • Event Handling: Adding dynamic event listeners, handling click events.

  • JavaScript Fundamentals: Primitive types, conditionals, loops, built-in methods like alert().

  • Client-side Storage: Local Storage methods: setItem(), getItem(), removeItem(), clear().

  • JSON Handling: JSON.stringify(), JSON.parse().

  • Array Methods: splice(), findIndex(), includes(), etc.
